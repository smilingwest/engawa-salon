# えんがわ茶論 公式Webサイト

> 学びと対話で未来を創る、教育×探究のプロフェッショナル

## 🎋 プロジェクト概要

**えんがわ茶論（ENGAWA SALON）**は、ワークショップデザイナー・ファシリテーターとして、学びと対話を通じて人と人、人と未来をつなぐ活動を行っています。

このWebサイトは、えんがわ茶論の専門性と実績をアピールし、訪問者との対話を促進するために作成されました。

### 主な特徴

- 🏯 **和モダンなデザイン** - 縁側や茶論のイメージを活かした温かみのあるUI
- 🤖 **AI対話型チャットボット** - 訪問者の質問に自動で応答
- 📱 **完全レスポンシブ** - スマホ、タブレット、PCすべてに対応
- 🎨 **キャッチーなビジュアル** - CSS/SVGで実装したイラストとアニメーション
- 🗺️ **インタラクティブな日本地図** - 活動エリアを視覚的に表示
- ⏱️ **タイムライン表示** - 10年間の実績を時系列で紹介

## 🎯 サイトの目的

1. **専門性のアピール** - レゴ®シリアスプレイ®、探究学習、SDGsなどの得意分野を明示
2. **実績の可視化** - 2016年から2025年までの50件以上の実績を紹介
3. **問い合わせ促進** - AIチャットとフォームで、気軽な相談を実現
4. **ブランディング** - 縁側のような温かく開かれた対話の場というイメージ確立

## 📧 お問い合わせフォームの仕組み

### Formspreeを使用した直接送信

このサイトでは、**Formspree**というサービスを使って、お問い合わせを直接メールで受け取れるようになっています。

### 動作の流れ

1. **訪問者がフォームに入力**
   - お名前、組織名、メールアドレス、電話番号、問い合わせ種別、内容

2. **「送信する」ボタンをクリック**
   - ボタンが「送信中...」に変わる
   - バックグラウンドでFormspreeに送信

3. **自動的に2通のメールが送信される**
   - ✉️ **あなた宛て**: `smilingwest@e-saron.com` に問い合わせ内容が届く
   - ✉️ **訪問者宛て**: 入力されたメールアドレスに確認メールが届く

4. **成功メッセージが表示される**
   - 「お問い合わせを送信しました！」
   - フォームがリセットされる

### 届くメールの内容

```
件名: 【えんがわ茶論】お問い合わせ - ワークショップ開催

name: 山田太郎
organization: ABC株式会社
email: yamada@example.com
phone: 090-1234-5678
type: ワークショップ開催
message: チームビルディングのワークショップを検討しています...
```

### 設定済みの内容

- **FormspreeフォームID**: `xeeqworz`
- **送信先メールアドレス**: `smilingwest@e-saron.com`
- **月間送信上限**: 50件（無料プラン）
- **スパム対策**: 有効
- **確認メール**: 自動送信

### メリット

✅ **ユーザーはワンクリックで送信完了** - メールソフト不要  
✅ **確実に届く** - サーバー経由で直接送信  
✅ **確認メール自動送信** - ユーザーに安心感  
✅ **スパム対策付き** - reCAPTCHA対応  
✅ **完全無料** - 月50件まで  
✅ **エラーハンドリング** - 送信失敗時の適切な通知
